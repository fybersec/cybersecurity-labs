# 🛡️ Cybersecurity-labs

Este repositorio es donde voy documentando los laboratorios de ciberseguridad que voy haciendo.

Más que solo resolver máquinas, la idea es entender qué está pasando en cada paso. Muchas veces los writeups que se encuentran son demasiado rápidos o van directo al resultado sin explicar el porqué. Aquí intento llevar el proceso con más calma y dejar claro el razonamiento detrás de cada cosa.

## 🎯 Enfoque

En cada laboratorio trato de seguir una lógica clara:

- Qué estoy buscando
- Qué encuentro
- Qué significa eso
- Cómo se puede aprovechar
- Qué se pudo haber hecho diferente

No es solo ejecutar comandos, es entender lo que está pasando.
